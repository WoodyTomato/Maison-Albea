# Maison Albéa - Landing Page Premium

Projet vitrine React + Tailwind CSS pour une marque fictive de produits ménagers écologiques premium destinée aux hôtels de luxe.

## Installation

```bash
npm install
npm run dev
```

Puis ouvrir l'URL locale affichée par Vite (généralement `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```
